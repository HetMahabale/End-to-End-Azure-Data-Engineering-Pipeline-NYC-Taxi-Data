# 🚀 End-to-End Azure Data Engineering Pipeline | NYC Taxi Data

## 📌 Project Overview

This project demonstrates a complete data engineering pipeline using Azure services.

## 🏗️ Architecture

* Azure Data Lake (Bronze Layer)
* Azure Databricks (Transformation)
* Delta Lake (Silver & Gold Layers)
* Azure Data Factory (Orchestration)

## 🔄 Pipeline Flow

1. Ingest raw data into Bronze layer
2. Transform data using Databricks
3. Store cleaned data in Silver layer
4. Create business-level tables in Gold layer

## 🛠️ Tech Stack

* Azure Data Factory
* Azure Data Lake Storage Gen2
* Azure Databricks
* Delta Lake
* PySpark

## 📊 Dataset

NYC Taxi Dataset

## 🚀 Key Features

* Scalable pipeline design
* Medallion architecture (Bronze/Silver/Gold)
* Real-world dataset
